Uptime monitor

Bobby Wibowo
2018-05-16

Hello there,

We now have uptime monitors for the safe and the hastebin that you can check out from here: https://stats.uptimerobot.com/OPwo4uOz8. It’s powered by Uptime Robot, so props to the folks in there.

I wanted to use status.fiery.me as its custom domain, but Uptime Robot seem to have issues with their certificates when being used with Cloudflare’s Full SSL (Strict), so I had no choice but to link the default URL in this blog post.

On a side note, i.fiery.me is the domain used to serve files uploaded to the safe. Unlike the safe which runs on Node, i.fiery.me runs on Nginx, which means either may die while not involving the other, so I made an extra monitor for it.

Anyways, that’s all. I hope you guys find it useful one way or another.

See ya.
